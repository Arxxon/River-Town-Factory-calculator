Небольшой калькулятор производства (надоело сидеть с кучей пометок в блокноте и несколькими открытыми калькуляторами). Пока реализовано примерно 1/3 рецептов (до синих капсул включительно).   
Рецепты будут добавляться по мере прохождения игры и открытия нового контента.    
Возможно после завершения всех рецептов будет локализация на Английский язык [EN].

Ссылка для веб версии - https://arxxon.github.io/River-Town-Factory-calculator/   
Ссылка на скачивания оффлайн варианта - https://github.com/Arxxon/River-Town-Factory-calculator/archive/refs/heads/main.zip   
Распаковать и открыть файл Index.html любым браузером (интернет не обязателен).

ВНИМАНИЕ, калькулятор считает рецепты только с 20% бонусом от таланта (возможность переключение на режим без таланта будет добавлен после полного прохождения игры, т.к. там оказывается далеко не 20%)

Калькулятор состоит из нескольких полей. 
В первом поле указывается желаемая вещь для производства.
Потом идет переключатель для смены режима отсчета, и после установки нужного режима можно указать количество желаемого количества предметов (в сутки, либо в минуту)
Справа есть окно, в котором указывается продолжительность игровых суток (минимальное значение в игре 13 минут, максимум 30 минут)

Сводная таблица с указанием суммарного количества заводов на все виды продукции, энергопотребления, а так же затрачиваемые изначальные ресурсы.

Интерактивная карта с возможностью маштабирования и переноса всех элементов в удобое расположение. 

На соединяющих линиях написано, сколько нужно передать предметов(и сколько это количество предметов будет делать заводов), актуально когда один ресурс идет в 2 и более крафтов.   
Желтыми цифрами написано производимое количество в минуту(если вдруг захочется вместо производства пересылать откуда то уже готовый ресурс)   
Зелеными цифрами написано количество заводов, которое смогут произвести желтое значение (если цифра дробная, то округлить в большую сторону)
